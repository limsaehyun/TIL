**UDP Header**

| Header                  | Description                                      |
| ----------------------- | ------------------------------------------------ |
| Source Port Number      | 송신 측의 응용 프로세스를 식별하기 위한 포트번호 |
| Destination Port Number | 수신 측의 으용 프로세스를 식별하기 위한 포트번호 |
| UDP Length              | 헤더와 데이터를 합친 길이                        |
| CheckSum                | 전송 중 세그먼트의 손성여부 확인                 |
